# PowerBI-MCP-AI-Data-Analysis
This project demonstrates how Claude AI can interact with Power BI through the Power BI MCP Server to automate parts of the data analysis workflow.  Instead of manually creating tables and relationships, AI is guided using prompts to perform common data modeling tasks.
AI-Powered Power BI Data Modeling using Claude + MCP

Built an AI-assisted Power BI workflow using Claude AI and the Power BI MCP Server.

Key Highlights

✔ Connected Claude AI with Power BI using MCP

✔ Imported Hospitality Dataset

✔ Generated Date Table using AI prompts

✔ Built Data Model

✔ Automatically identified matching keys and created relationships

✔ Reduced manual setup time using AI-assisted automation
